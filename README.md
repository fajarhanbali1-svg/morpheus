# Morpheus - Digital Organism

Program yang menulis ulang dirinya sendiri dari Python ke JavaScript dan sebaliknya, membawa DNA (hash + misi) di setiap metamorfosis.

## Cara Menjalankan
Pastikan Python 3 terinstal.
```bash
python3 morpheus.py
```

Jika Node.js juga terinstal, program akan otomatis melanjutkan metamorfosis ke JavaScript, lalu kembali ke Python, dan seterusnya.

Peringatan

· Jangan jalankan di folder yang berisi file penting, karena program menghasilkan file morpheus.js dan morpheus_next.py di direktori yang sama.
· Untuk menghentikan rantai, tekan Ctrl+C atau hapus file turunan.
· Kode ini adalah bukti konsep, bukan untuk produksi.
